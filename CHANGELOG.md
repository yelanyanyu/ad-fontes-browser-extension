# 更新日志 (Changelog)

## [1.1.1] - 2026-01-26

### ✨ 新增功能
- **实时预览**：在生成按钮下方新增了预览区域，点击生成后可立即查看格式化后的卡片内容。

### 🐛 修复与优化
- **构建配置修复**：修复了 `vite.config.js` 的入口路径错误，解决了 Edge 浏览器报错 "The default_popup file in the manifest doesn't exist" 的问题。
- **项目结构清理**：移除了根目录下冗余的 `src/popup.js` 文件，规范了项目结构。
- **暂存功能优化**：修复了输入内容无法保存的问题，增加了防抖 (Debounce) 处理，并确保在 DOM 加载完成后才初始化存储逻辑。
- **NLP 优化**：修复了部分单词（如 "appropriated"）无法正确转换为词根（Lemma）的问题，现在能准确获取原型（如 "appropriate"）并用于词典查询。
- **输出格式优化**：生成的卡片现在会使用词根（Lemma）作为标题，而不是用户输入的原始形式。

## [1.0.0] - 2026-01-26

### 🎉 初始版本发布
- **词典查询**：利用 Free Dictionary API 实时获取单词释义、音标和词性。
- **上下文卡片**：生成包含原始上下文句子和自定义笔记的结构化单词卡。
- **智能词根化**：利用 NLP 技术自动将单词转换为原型（例如 "took" → "take"）。
- **一键复制**：自动格式化并复制生成的卡片到剪贴板，方便粘贴到 Anki 或笔记软件中。
- **跨浏览器支持**：验证并支持 Chrome 和 Microsoft Edge 浏览器。
